# Loja Mágica ✨

Sistema de terminal que permite navegar por categorias (livros, artefatos, poções e itens), buscar e adicionar novos itens.

## Como executar
Clone o projeto e rode com:
`dart run`

## Funcionalidades
- Menu interativo
- Busca por item
- Adição de novos itens
- Modularização com funções
